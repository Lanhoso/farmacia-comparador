# Comparador de Precios de Farmacias — Chile

A web scraper and price comparison tool for Chilean pharmacy chains.

## Background

In 2009, Chile's three major pharmacy chains — **Cruz Verde**, **Salcobrand**, and **Ahumada** — were convicted of coordinating drug price increases, affecting hundreds of essential medications. Despite this conviction, no publicly available tool exists to compare prescription drug prices across pharmacies in real time.

This project fills that gap.

## What It Does

- Scrapes drug prices directly from pharmacy websites using Playwright (handles JS-heavy Angular frontends)
- Stores results as structured JSON in `/data`
- Renders a clean comparison table in a self-contained HTML page (no backend required)
- Currently covers **Cruz Verde**; Salcobrand and Ahumada coming next

## Installation

**Requirements:** Python 3.9+

```bash
# 1. Clone the repo
git clone https://github.com/your-username/farmacia-comparador.git
cd farmacia-comparador

# 2. Install dependencies
pip install -r requirements.txt

# 3. Install the Chromium browser Playwright uses
python -m playwright install chromium
```

## Usage

### Scrape a drug

```bash
# Default: Metformina 850mg
python scraper.py

# Any other drug
python scraper.py "Ibuprofeno 400mg"
python scraper.py "Atorvastatina 20mg"
```

Results are saved to `data/<drug_slug>.json` and embedded into `index.html` automatically.

### View results

Open `index.html` directly in your browser — no server needed. The latest scraped data is embedded inline.

Or serve it locally:

```bash
python serve.py
# → http://localhost:8000
```

### Debug layout changes

If the scraper stops finding results (the site may have updated its CSS), run the debug tool to inspect the live DOM:

```bash
python debug_scraper.py "Metformina 850mg"
```

This opens a visible browser window and prints all elements containing prices, so you can update `SELECTORS` in `scraper.py`.

## Output Format

Each JSON file in `data/` follows this schema:

```json
{
  "query": "Metformina 850mg",
  "generated_at": "2026-03-30T19:08:00+00:00",
  "count": 12,
  "results": [
    {
      "pharmacy": "Cruz Verde",
      "drug_name": "Metformina 850 mg 30 Comprimidos",
      "price_clp": 2712,
      "original_price_clp": 3390,
      "url": "https://www.cruzverde.cl/search?query=Metformina%20850mg",
      "product_url": "https://www.cruzverde.cl/metformina-850-mg-30-comprimidos/270505.html",
      "scraped_at": "2026-03-30T19:08:00+00:00"
    }
  ]
}
```

## Legal & Ethics

This tool scrapes **publicly accessible** product listing pages — the same pages any user visits when browsing the pharmacy website. No login, no private data, no account required. Requests are rate-limited and made at human-scale frequency.

The goal is **consumer transparency**, not commercial data harvesting.
